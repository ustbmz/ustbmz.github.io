rem元素​

  • em : 父元素 font-size

  • rem : 根元素 font-size

rem动态换算​

页面可视区域分为 100vw 与 100vh

  • vw 使用vw动态换算rem
html{
  /* 100px */
  font-size: 5.208333vw;
}
body{
  font-size: .16rem;
}
.main {
  height: 1rem;
  width: 1rem;
  background-color: pink;
}
